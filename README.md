# SSUSTAGRAM
길벗출판사 node.js 교과서 참고해서 만들었습니다.

사용한 기술스택은 다음과 같습니다.

Back-End : MySql, NodeJS, Express, Sequelize

Front-End : HTML, CSS, JavaScript
