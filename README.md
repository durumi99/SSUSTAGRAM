# SSUSTAGRAM
길벗출판사 node.js 교과서 참고해서 만들었습니다.

사용한 기술스택은 다음과 같습니다.

Back-End : MySql, NodeJS, ExpressJs, Sequelize

Front-End : HTML, CSS, JavaScript, nunjucks, jQuery

Tool : Git


구현 내용은 다음과 같습니다

초기화면 (/) 

사용자 등록 (/account) 

사용자 소개 (/profile) 

홈 (/home) 

검색(/search)

업로드 (/new) 

편집 (/edit)  

팔로우 목록(/follow)
